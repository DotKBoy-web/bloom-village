سياسة خصوصية Bloom Village

تاريخ السريان: 22 يونيو 2026

تم تصميم Bloom Village كلعبة حديقة بسيطة وآمنة للعائلة وتعمل دون الحاجة إلى حساب.

جمع البيانات

لا تجمع Bloom Village أي بيانات شخصية من المستخدمين.

  • لا يوجد تسجيل دخول.
  • لا يتم جمع الاسم أو البريد أو رقم الهاتف أو الموقع أو الصور أو المعرفات.
  • لا توجد إعلانات أو أدوات تتبع أو تحليلات.
  • لا يتم بيع أو مشاركة أي بيانات مع المعلنين.

تقدم اللعبة المحلي

يتم حفظ تقدم اللعبة محلياً على الجهاز فقط، مثل الزهور والعملات والأثاث واللغة المختارة. لا يتم إرسال هذه البيانات إلى المطور أو أي طرف ثالث.

الأطفال

اللعبة مصممة لتكون لطيفة وآمنة للأطفال والعائلات، ولا تحتوي على إعلانات أو مشتريات داخل التطبيق أو دردشة أو محتوى ينشئه المستخدمون.
