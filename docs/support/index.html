Bloom Village Support

Need help with Bloom Village? This page provides basic support information for players and families.

Common Questions

Does Bloom Village require sign-in?

No. Bloom Village does not require an account, username, password, or email sign-in.

Does Bloom Village collect personal data?

No. Bloom Village does not collect personal data.

Does Bloom Village have ads?

No. Bloom Village does not show third-party ads.

Does the app work offline?

Yes. Bloom Village is designed as a simple offline game experience.

Where is game progress saved?

Game progress is saved locally on the device.

Which languages are supported?

Bloom Village supports English and Arabic.

Troubleshooting

If the app does not open correctly, try closing it completely and opening it again. If the issue continues, restart the device and try again.

Contact Support

Email: dotkboy@outlook.com

When contacting support, please include the device type, iOS/iPadOS version, and a short description of the issue.
